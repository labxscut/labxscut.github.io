DeepLB API Documentation

DeepLB is a mixed-language project (shell + R + Python + C/C++). This docs section provides dedicated API references for Python and R interfaces.

Python API

Thin compatibility wrapper (`deeplb`) around the existing shell pipeline, plus CLI entrypoints.

R API
