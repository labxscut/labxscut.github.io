# DeepLB API Documentation

This section documents DeepLB interfaces as two API surfaces:

- [Python API](./PYTHON_API.html)
- [R API](./R_API.html)

DeepLB is a mixed-language project (shell + R + Python + C/C++).

- Python API is currently a compatibility wrapper around the existing shell pipeline.
- R API is currently script-oriented (`Rscript ...`) rather than a packaged R namespace.
