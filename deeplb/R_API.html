R Script Interfaces

DeepLB R API

DeepLB currently exposes R functionality as script interfaces invoked with Rscript.

Runtime model

  • Execute via Rscript <script> <args...>
  • Arguments parsed by commandArgs(trailingOnly = TRUE)
  • Orchestrated by Scripts/DeepLB_pipeline.sh

Script contracts

1.1_reformat_TCGA.R

Args: current_path, tumor_type

Rscript Scripts/Part1.Marker_Selection/1.1_reformat_TCGA.R /path/to/DeepLB LIHC

1.1_prepare_sample_annotation.R

Args: current_path, meta, tumor, bgfile

Rscript Scripts/Part1.Marker_Selection/1.1_prepare_sample_annotation.R /path/to/DeepLB all_WGBS_sample_metadata.xlsx LIHC background_for_train.txt
